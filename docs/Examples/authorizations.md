# Authorization

Authorization is necessary to verify whether a customer's payment source has sufficient funds.It certainly establishes validity when assessing debits against a particular card or bank account.It can be used in both one-time and recurring payment transactions.

---

### Endpoint
<!-- theme: success -->
>**POST** `/v1/authorization/search`
